# PS1 Company Explorer

A full-stack web app for BITS Pilani students to search and explore companies coming for Practice School 1 placements.

- **File Upload** вЂ” drag-and-drop Excel (.xlsx, .xls) or CSV upload
- **Live File Watcher** вЂ” auto-reloads data when the watched file changes (via chokidar + SSE)

- **Full-text Search** вЂ” across company name, domain, city, and project details with highlights

- **Responsive** вЂ” works on mobile and desktop

## Setup

### Prerequisites

- Node.js 18+
- npm or pnpm

### 1. Install dependencies

```bash
# Backend
cd backend
npm install

# Frontend (in a separate terminal)
cd frontend
npm install
```

### 2. Configure environment (optional)

```bash
cp backend/.env.example backend/.env
```

Edit `backend/.env`:

```env
PORT=3001
# Optional: path to an Excel/CSV file to watch for auto-reload
WATCH_FILE_PATH=../data/companies.xlsx
```

### 3. Run the app

In two terminals:

```bash
# Terminal 1 вЂ” Backend
cd backend
npm run dev

# Terminal 2 вЂ” Frontend
cd frontend
npm run dev
```

Open **http://localhost:5173** in your browser.

## Data Format

The app accepts Excel (.xlsx, .xls) or CSV files. Column names are detected flexibly вЂ” these headers work:

| Required | Example column names |
|---|---|
| Company Name | `Company Name`, `Company`, `Organization` |
| Domain | `Domain`, `Field`, `Branch` |
| City | `City`, `Location`, `HQ` |
| Project Details | `Project Details`, `Project`, `Description`, `JD` |

Any additional columns are preserved and shown in the company detail modal.

### Sample template

Generate a sample file with:

```bash
cd "PS1 app"
node create_sample.js
```

This creates `sample_data.xlsx` with 12 example companies.

---

## File Watcher

If `WATCH_FILE_PATH` is set in `.env`, the backend watches that file and auto-reloads when it changes. A **Server-Sent Events** stream pushes the update to all connected browsers instantly.

If you don't want auto-watch, just use the **Upload** button in the UI or the **Refresh** button to re-parse the last uploaded file.
